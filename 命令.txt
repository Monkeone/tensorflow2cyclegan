默认情况下，您可以将数据放入数据/男性女性/目录。培训和测试数据应在4个单独的目录中提供，如下所示：

训练用男性面部图像
培训B-训练中的女性面部形象
测试用男性面部图像
测试B.测试用女性面部图像
将代码运行为

python train.py
如果数据位于不同的目录中，则可以在运行时使用-数据dir国旗：

python train.py --data_dir <path/to/data>
若要查看可更改的参数，请运行

python train.py -h
培训损失将在日志文件夹中更新，该文件夹可以与张卡一起运行，以便在浏览器上可视化生成器和判别器丢失。运行以下命令：

tensorboard --logdir=logs
在训练结束时，该代码的输出是两个发电机和两个判别器的模型权重。这些将保存为：

发电机AToB.h5-43.5MB
发电机BToA.h5-43.5MB
鉴别器A.h5-10.5MB
鉴别器B.h5-10.5MB
测试
一旦对模型进行了培训，模型文件应该与具有名称的测试脚本位于同一个文件夹中：发电机AToB.h5和发电机BToA.h5..测试数据应该在-数据dir参数在文件夹结构中的参数，如上文所述。使用-批号标志，您可以指定主页许多测试图像应该被修改。

您可以通过运行以下命令来测试模型：

python test.py
结果将在结果文件夹。创建两个文件夹M2F和F2M在结果文件夹中。相应的转换将出现在这两个文件夹中。结果将包括假的，真实的，重建的和身份的图像。
